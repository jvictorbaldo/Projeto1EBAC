# Projeto 1 - EBAC

## Este projeto é uma calculadora em Python que realiza operações matemáticas básicas, como adição, subtração, multiplicação e divisão. Inclui um arquivo em shell script que automatiza a execução da calculadora, permitindo que os usuários iniciem o programa rapidamente, sem precisar rodar o código manualmente. Ideal para facilitar cálculos do dia a dia.
